/** brew install node **/
One very convenient way to install Node.js is through a package manager. In this case, every operating system has its
own.
On macOS, Homebrew is the de-facto standard, and — once installed — allows to install Node.js very easily.

/** nvm **/
nvm is a popular way to run Node.js. It allows you to easily switch the Node.js version, and install new versions to
try and easily rollback if something breaks,
